Web2Unity Controller

I thought my phone would make a good controller for my unity game. Turned out that people don't do that too often, so I made it happen. I developed a web-server to handle touch events from the phone and host a controller website, then I set up a system (via websockets) that allowed the unity game to register commands of the server.

VR Musical Compass

I was developing a UI for a VR Diving Simulator in a team of four with Unity3D. To develop a real 'eyes-free' interface, (you might not want to block your vision with a clunky heads-up or arm-tracked user interface), we came up with a number of features to enable eyes free navigation. Audio warnings when oxygen and battery is low, or when the pressure is to high, or when calling. But by far my favorite part is the Musical Compass. It features:

  • Dynamic change of music based on distance to target
  • Dampening of music depending on whether you are above or under water
  • Music gets distorted and distant when you turn away from the target, and clear when you turn towards it .

All coming together to navigate the Diver through this underwater adventure.



Vocal Pong

I don't have to explain Pong, but this version lets you play blindly and eyes free. The game-state is completly sonified (you can hear the position of the ball and where it will land) and you play by matching pitch. Additionally it features a fully vocalized menu that uses speech-recognition to access different functions such as reading out the score and instructions. Developed in Python.

AudioMaze

[in development] Labyrinth game with completely sonified game state that guides the player through a labyrinth. Aim of the game is for the player, to solve a puzzle by moving objects. But there is a trick to this, the player will never see where the objects belong. Instead the object is a musical section, and moving it to the right place will make the section 'lock in to place'. Together with other objects the player can build up an entire musical piece, by putting them in the right places.

Video Graph

An interactive Video player developed in Unity3D. I developed this for the Uchicago Human-Robot-Interaction-Lab with Sarah Sebo, to help conduct user-studies. It allows you to set up several different 'choose your own adventure' video sequences, where you can chose which video is played next while another video is playing. The goal is to minimize the delay between videos by loading the different possible next videos concurrently, it also allows for filler videos that wait for user input all without creating noticable breaks. It uses a graph structure to organize which videos can follow on which videos, hence the name.

Atten Java Plugin

A VST-Effect plugin to be loaded into a Digital Audio Workstation (DAW). The Programm distorts the sound (crunchy rock guitar sound) and adds a delay (echo effect). Beyond that it also features a bunch of other parameters that the user can tweak to customize their sound. The plugin is programmed in Java. The translation from Java to the VST-format is done using the JVSTWrapper library.
